Pastelería Martti

by Pau Martino

Proyecto: Pastelería Martti

Este proyecto se basa en el desarrollo web de la Pastelería Martti. En esta primera entrega, se simula la búsqueda de productos disponibles a través de la consola de JavaScript y ventanas emergentes como prompt, alert y confirm, permitiendo al usuario buscar productos por nombre o categoría y decidir si desea continuar la búsqueda o cancelarla.

En futuras entregas, se incorporará la funcionalidad de carrito de compras, selección de cantidades y confirmación del pedido, con redirección a WhatsApp para finalizar la compra, ya que la pastelería no cuenta con stock fijo y este medio facilita la coordinación de los pedidos.

Conocimientos aplicados en esta entrega:

  • Estructura de archivos HTML y aplicación de CSS
  • Vinculación de HTML a JavaScript
  • Uso de objetos
  • Arrays: declaración de arrays y aplicación de métodos
  • Funciones para manejar la lógica de búsqueda
  • Prompts para interactuar con el usuario y obtener consultas de búsqueda
  • Alerts para mostrar resultados de búsqueda y mensajes de bienvenida y despedida
  • Confirm para preguntar al usuario si desea continuar la búsqueda
  • Operadores lógicos para controlar el flujo de búsqueda y decisiones
  • Condicionales para mostrar mensajes según resultados
  • Bucles para permitir múltiples búsquedas hasta finalizar
