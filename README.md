Pastelería Martti

Proyecto frontend de e-commerce desarrollado con HTML5, SCSS y JavaScript.

El sitio representa una pastelería online con navegación por categorías, buscador categoría o productos y sistema de carrito de compras.
Funcionalidades:
- Navegación por categorías de productos
- Buscador por nombre de producto o categoría
- Sistema de carrito de compras

Se utilizó:
- HTML5 
- SCSS 
- JavaScript 
- Bootstrap
